# VV — VALERY VISUALS

Сайт для ТОО "VALERY VISUALS"

## Стек

- **React 19 + TypeScript 5.9.3**

React используется как основа UI: компонентная структура, состояние, маршруты, переиспользуемые секции страницы.
TypeScript добавляет строгую типизацию пропсов, данных и хуков, чтобы раньше ловить ошибки и проще поддерживать код при росте проекта.

- **Vite 7**

Инструмент сборки и dev-сервер, который даёт быстрый локальный запуск, HMR и оптимизированную продакшн сборку.

- **Tailwind CSS 4.2**

Utility-first подход для быстрой адаптивной верстки и единообразной дизайн-системы без большого объёма ручного CSS.

- **GSAP 3.14.2 (+ ScrollTrigger) + Lenis 1.3.19 (скролл‑сцены)**

GSAP отвечает за плавные и точные анимации.
ScrollTrigger связывает анимации со скроллом.
Lenis обеспечивает более плавную прокрутку и улучшает восприятие скролл-сцен.

- **Motion 12.38.0. (микро‑анимации/модалки/hover)**

Используется для микро-анимаций интерфейса, hover-состояний, появления/исчезновения элементов и модалок.

- **React Router 7 (`/`, `/portfolio`)**

Обеспечивает клиентскую маршрутизацию между страницами без полной перезагрузки и поддерживает SPA-подход.

## Быстрый старт

```bash
npm install
npm run dev
```

## Команды

```bash
npm run dev      # dev-сервер
npm run build    # сборка в dist
npm run preview  # предпросмотр сборки
```

## Структура проекта (коротко)

- `src/main.tsx` — вход, роутинг, оверлеи
- `src/App.tsx` — главная страница (блоки preloader → hero → about → portfolio → contact → footer)
- `src/components/` — блоки, секции/сцены сайта
- `src/pages/PortfolioWorksPage.tsx` — страница `/portfolio`
- `src/data/worksProjects.ts` — данные карточек работ
